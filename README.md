# jsplumb-example

运行步骤：
## npm install 
如果出现安装node-sass提示没有vendor目录，解决办法：
在node_modules/node-sass目录下面新建一个vendor的空目录，然后运行npm rebuild node-sass --save-dev即可


## npm run dev



> Use Jsplumb draw flow chart
